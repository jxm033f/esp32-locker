# Module 2: Locker
COMS BC 3930

Through this module, I created an interactive locker that allows you to unlock a secret box.

## Materials:
- Raspberry Pi (used for Power)
- ESP32 (runs Arduino program)
- 1 analog joystick
- 2 large buttons
- 1 SPST switch
- 1 LEDPixel
- 1 Servo
- 1 PWM
